## What are Pantone Colors of the Year?

Since 2000, Pantone has selected a **Color of the Year** that represents global trends, culture, and the spirit of the times. Each color tells a story and evokes specific emotions.

## How to Use Pantone Themes

### Method 1: Theme Selector (Recommended)
1. Open the **Theme dropdown** in the header
2. Scroll to **"Pantone Color of the Year"** section
3. Select your favorite year's color
4. Theme applies instantly!

### Method 2: JavaScript Console
```javascript
// Set a Pantone theme
modeler.setTheme('viva-magenta');  // 2023
modeler.setTheme('very-peri');      // 2022
modeler.setTheme('classic-blue');   // 2020
// ... etc
```
